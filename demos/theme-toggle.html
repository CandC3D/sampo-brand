Theme toggle.sampo-theme-toggle

Moon-glyph button. Cycles system → light → dark → system. Persists to localStorage under 'sampo.theme'. JS lives in @sampo/brand's theme.js.

Use the toggle in the demo header above — it's the same component.

Current theme: 

<button class="sampo-theme-toggle" id="themeToggle">
  <span class="sampo-theme-toggle-glyph"></span>
</button>

window.SampoTheme.initThemeToggle({
  buttonSelector: '#themeToggle',
  onChange: () => { /* re-render theme-sensitive SVGs here */ }
});
