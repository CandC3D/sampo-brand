Buttons.btn / .btn-*

Base button plus variant modifiers. Stack .btn-dashed-hover on top of any variant for the mill's dashed-ring hover vocabulary.

<button class="btn">Default</button>
<button class="btn primary">Primary</button>
<button class="btn btn-ghost">Ghost</button>
<button class="btn btn-cta">CTA</button>
<button class="btn btn-dashed-hover">Dashed on hover</button>

The .btn-cta pumpkin fill is reserved for the site's one-per-page primary action. Ghost and dashed-hover stack freely; use them when the interaction shouldn't draw the eye.
