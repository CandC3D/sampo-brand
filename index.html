SAMPO @SAMPO/BRAND

@sampo/brand

Shared visual system for the Sampo Diagnostic Kit. Palette, typography, mill brand mark, component primitives. One source of truth for the generator and the public site.

Component demos → Source on GitHub →

How to consume

Plain <link> and <script> tags — no bundler required:

<link rel="stylesheet" href="https://candc3d.github.io/sampo-brand/src/fonts.css">
<link rel="stylesheet" href="https://candc3d.github.io/sampo-brand/src/tokens.css">
<link rel="stylesheet" href="https://candc3d.github.io/sampo-brand/src/themes.css">
<link rel="stylesheet" href="https://candc3d.github.io/sampo-brand/src/components.css">
<link rel="stylesheet" href="https://candc3d.github.io/sampo-brand/src/motion.css">
<script src="https://candc3d.github.io/sampo-brand/src/mill.js"></script>
<script src="https://candc3d.github.io/sampo-brand/src/theme.js"></script>

What ships

  • mill.js — brand compositor (pure SVG strings)
  • theme.js — getTheme, applyTheme, initThemeToggle
  • tokens.css — palette, type scale, spacing, radii
  • themes.css — light/dark overrides
  • components.css — .sampo-lockup, .sampo-divider, .sampo-corner-mark, .sampo-watermark, .preview-card, .sampo-toast, .sampo-theme-toggle, .btn variants
  • motion.css — prefers-reduced-motion policy
  • fonts/ — self-hosted Instrument Sans + Instrument Serif (6 woff2, ~112 KB)
